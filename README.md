# DailyBloom

DailyBloom is a preschool management platform designed to simplify school administration, parent communication, learner tracking, and operational workflows through a mobile-friendly and low-data experience.

## Features

- Daily learner summaries
- Parent communication
- Attendance tracking
- Payment reminders
- Teacher activity management
- School dashboards

## Target Users

- Preschool owners
- Principals
- Teachers
- Parents and guardians

## Technologies

- Next.js
- Supabase
- Vercel
- GitHub
